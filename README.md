# Express.js framework: Note Taker

> Assignment 11, Boot Camp
> University of Washington, Full Stack Development

Create a note taker application that will use an express backend and save and retrive the note data from a JSON file.  

## User Story 

```
AS A student
I WANT to take notes and save them for later
SO I CAN access them and take keep track of what I wrote down

```

## Acceptance Criteria 

```
User will access note taker on heroku app
User will write out their notes
Notes are saved in the app
User can access saved notes 
User has the option to delete saved notes

```
